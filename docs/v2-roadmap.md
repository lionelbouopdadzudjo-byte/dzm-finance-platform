# V2 Modular Roadmap

Le backend expose des modules V2 stubs dans `backend/src/modules/v2`:
- `natural-language-search.module.ts`
- `smart-timeline.module.ts`
- `collaborative-comments.module.ts`
- `voice.module.ts`
- `forecast.module.ts`
- `contextual-recommendations.module.ts`
- `advanced-allocation-matching.module.ts`

Chaque module expose interfaces + service injectable pour évoluer sans casser le MVP.
